# 📝 Streamlit Text Editor (Dockerized)

A simple **Text Editor Web App** built with **Streamlit** and packaged with **Docker**.  
This app lets you type, view word counts, and download your text as a `.txt` file.

---

## 🚀 Features
- Simple text editor with live word count
- Download your content as a `.txt` file
- Runs inside a Docker container
- Lightweight and easy to deploy
